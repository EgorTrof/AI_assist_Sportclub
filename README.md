ИИ-ассистент для Спортивного клуба единоборств — это интеллектуальный Telegram-бот, созданный для автоматизации рутинных задач спортивного клуба: консультирование клиентов, ответы на частые вопросы и запись на тренировки.
Бот использует мощь больших языковых моделей (Qwen) и платформу Savvy для создания персонализированного диалога, а также интегрируется с Google Calendar для автоматического планирования занятий.

Цели проекта:
  ✅ Снизить нагрузку на администраторов клуба
  ✅ Обеспечить круглосуточную поддержку клиентов
  ✅ Упростить процесс записи на тренировки
  ✅ Предоставить точные ответы на вопросы по расписанию, ценам и правилам клуба
✨ Возможности:
  💬 Умный диалог
Осмысленное общение с пользователем на естественном языке
Контекстная память в рамках сессии
Поддержка русского языка с учётом спортивной терминологии
  📚 База знаний
Ответы на вопросы по:
Расписанию тренировок
Стоимости абонементов
Правилам клуба и экипировке
Тренерскому составу
Частым вопросам (FAQ)
Возможность обновления базы знаний без перезапуска бота
  🗓️ Запись на тренировку
Интерактивный сценарий записи: выбор направления, даты и времени
Проверка доступности слотов в реальном времени
Автоматическое создание события в Google Calendar
Отправка подтверждения пользователю в Telegram
  🔔 Уведомления
Напоминания о предстоящей тренировке (за 24 часа и за 1 час)
Информирование об изменениях в расписании
