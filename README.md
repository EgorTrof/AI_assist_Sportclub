**ИИ-ассистент для Спортивного клуба единоборств** — это умный чат-бот в Telegram, созданный для повышения качества обслуживания клиентов и автоматизации рутинных задач администратора.

Бот использует мощь больших языковых моделей (Qwen) и платформу Savvy для создания персонализированного диалога, понимает контекст беседы, отвечает на частые вопросы и помогает записаться на тренировку в один клик.

### 🎯 Цели проекта

- ✅ Снизить нагрузку на администраторов за счёт автоматизации ответов
- ✅ Обеспечить круглосуточную поддержку клиентов
- ✅ Упростить процесс записи на тренировки
- ✅ Интегрировать расписание с Google Calendar
- ✅ Предоставить персонализированный опыт общения

---

## ✨ Возможности

### 💬 Умный диалог
- Осмысленное общение на естественном языке благодаря нейросети Qwen
- Поддержка контекста в рамках сессии
- Адаптация стиля общения под аудиторию клуба

### 📚 База знаний
- Ответы на часто задаваемые вопросы:
  - Расписание занятий
  - Стоимость абонементов
  - Требования к экипировке
  - Информация о тренерах
  - Правила посещения зала
- Возможность динамического обновления базы знаний

### 🗓️ Запись на тренировку
- Интерактивный сценарий записи: выбор направления, даты, времени
- Проверка доступности слотов в реальном времени
- Автоматическое создание события в **Google Calendar**
- Отправка подтверждения и напоминаний клиенту
